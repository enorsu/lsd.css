# LSD(the drug).css

A css add-on to make your website as unappealing as possible while remaining usable.

> Note: LSD.css is recommended to be used with existing styling(eg. [PicoCSS](https://picocss.com)), it does not provide any other styling than random colors

[Preview with PicoCSS](https://enorsu.github.io/lsd.css//DEMO.html?1=lsd.css&2=https://cdn.jsdelivr.net/npm/@picocss/pico@2/css/pico.min.css)(flash warning)

Quick install:

Add this to your website's <head>(below all of the other entries):

`<link rel="stylesheet" href="https://enorsu.github.io/lsd.css/lsd.min.css">`
