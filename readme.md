# LSD(the drug).css

A css add-on you to make your website as unappealing as possible while remaining usable.

[Preview](https://enorsu.github.io/lsd.css/DEMO.html)(flash warning)

Quick install:

Add this to your website's <head>(below all of the other entries):

`<link rel="stylesheet" href="https://enorsu.github.io/lsd.css/lsd.css">`
